# 板卡环境记录

本目录记录 Tang Nano 20K 的公开环境确认信息，用于说明项目的硬件、工具链和串口基础条件。个人环境截图、原始日志和临时过程材料不放入本目录。

## 当前板卡状态

- 板卡：Sipeed Tang Nano 20K
- FPGA：`GW2AR-LV18QN88C8/I7`
- USB 桥接/板载下载器：BL616
- 串口终端：可使用任意 115200 8N1 终端工具
- 上电结果：板卡可以正常上电
- 当前 FPGA/SoC 程序：板卡可通过串口输出文本，说明基础串口链路可用
- Gowin 器件选择：已确认可选 `GW2AR-LV18QN88C8/I7`，显示为 `GW2AR-18C`、`QFN88`、`C8/I7`

关键结论是板卡能够上电，且 BL616 下载/串口桥接链路可用于后续 FPGA 工程验证。
